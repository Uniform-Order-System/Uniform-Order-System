WhatsApp Setup

Connect your WhatsApp Business number so orders arrive here automatically.

  1. Create a Meta Business account and a WhatsApp Business App at developers.facebook.com.
  2. Get a permanent access token and your Phone Number ID from the WhatsApp > API Setup page.
  3. Put them in this server's .env file as WHATSAPP_TOKEN and WHATSAPP_PHONE_NUMBER_ID.
  4. Set the webhook URL in Meta to https://YOUR-DOMAIN/webhook, with verify token matching WHATSAPP_VERIFY_TOKEN in .env.
  5. Ask customers to message your new WhatsApp Business number. Every message becomes an order here.

Full step-by-step is in the README that came with this project.
